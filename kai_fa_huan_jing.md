# 开发环境

### [node](https://nodejs.org/en/)

### [nvm](https://github.com/creationix/nvm)
使用nvm管理node, nvm安装
```
curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.31.0/install.sh | bash
```
使用nvm安装node.js 4.4.0 LTS
```
nvm install 4.4.0
```

### [npm](https://github.com/npm/npm)
